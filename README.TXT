BioTrack Laboratory Management System
=====================================

Version: 1.0.0

SYSTEM REQUIREMENTS:
- Java Runtime Environment (JRE) 8 or higher
- Windows 7/8/10/11

INSTALLATION:
1. Run 'install.bat'
2. Follow the on-screen instructions
3. Launch BioTrack from desktop shortcut

DEFAULT LOGIN:
Username: admin
Password: admin123

FEATURES:
- Equipment inventory management
- Borrow/Return tracking
- Transaction history
- User authentication
- Data export

SUPPORT:
For issues, check if Java is installed:
- Open Command Prompt
- Type: java -version
- Should show Java version 1.8 or higher

DATA LOCATION:
User data is stored in: C:\Users\[YourUsername]\BioTrack\data\
